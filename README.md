Upload the entire sleep-edf folder to Google Drive and run
sleep-edf/final_model/final_model_colab.ipynb in Google Colab.

The notebook contains the full training pipeline, including data loading, preprocessing, Stage 1 training, Stage 2 training, and inference/evaluation.
For reproducibility, the trained checkpoint files are included in final_model/checkpoints, and the reported results were generated using these checkpoints.
